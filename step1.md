# Step 1: 扫码商品录入系统

## 🎯 阶段目标

实现扫码枪扫描条形码自动识别商品，并使用 JSON 文件存储商品信息。

---

## 📋 功能概述

| 功能模块 | 描述 |
|---------|------|
| 条形码扫描 | 通过扫码枪读取商品条形码 |
| 商品识别 | 将条形码映射为具体商品信息 |
| 数据存储 | 使用 JSON 文件持久化存储商品数据 |

---

## 🏗️ 系统架构

```
扫码枪 → 条形码解析 → 商品匹配 → JSON存储
```

---

## 📦 商品数据结构

```json
{
  "products": [
    {
      "barcode": "6901234567890",
      "name": "可口可乐 330ml",
      "category": "饮料",
      "price": 3.50,
      "stock": 100,
      "unit": "瓶",
      "created_at": "2024-01-15T10:30:00Z",
      "updated_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```

---

## ✅ 验收标准

- [ ] 扫码枪能够正常读取条形码
- [ ] 条形码与商品信息正确匹配
- [ ] 商品数据成功保存到 JSON 文件
- [ ] 支持商品信息的增删改查操作
- [ ] 数据格式规范，易于后续扩展
